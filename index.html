BSP Messenger System

About

Why would we build the system?

  • Automation as much as possible
  • Less time on training
  • Easier accessibility
  • Greater centralization
  • Faster Production

What it can do?

  • COCREE Converter

    The COCREE Converter can Find Excel, Find XSD, Convert Excel. The Program will ask the user where the save file location is. A Text Box that provides prompts for the user If task is successful.

  • COCREE Validator

    The COCREE Validator features are Find XSD, XML validation, Logical validation. This will lessen the margin of error when submitting the XML file to BSP.

  • FRP 15 Converter

    The FRP15 Converter can Find Excel, Find XSD, FRP 15 Excel. The Program will ask the user where the save file location is.

  • FRP 15 XML Validation

    The FRP 15 Validator features are Find XSD, XML validation, This will lessen the margin of error when submitting the XML file to BSP.

  • API- Machine to Machine XML submission from Bank to BSP

    The API - M2M Bank to BSP will have a Submission of XML, Download Validation Report, Download Submission Confirmation. This will automate the process of submitting the XML file.
